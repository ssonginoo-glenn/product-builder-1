
# Lotto Number Generator

## Overview

This application generates a set of 6 unique random numbers for a lottery drawing. The user can click a button to generate a new set of numbers.

## Project Outline

### Style and Design
*   **Layout:** Centered, responsive layout.
*   **Color Palette:** A modern and energetic color scheme with a wide range of hues.
*   **Typography:** Expressive and clear fonts to enhance readability.
*   **Effects:**
    *   Subtle noise texture on the background for a premium feel.
    *   Multi-layered drop shadows for depth.
    *   "Glow" effect on interactive elements.
*   **Iconography:** Icons to improve usability.

### Features
*   Generates 6 unique random numbers between 1 and 45.
*   Displays the generated numbers.
*   Button to trigger the number generation.
